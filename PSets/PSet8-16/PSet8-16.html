Problemas

Implemente Mashup en pset8/mashup/.

Consejos

  • Asegúrese de usar Python 3, no Python 2.
  • Todo el código de Python que escriba debe ser coherente con la Guía de estilo para el código de Python , también conocida como PEP 8.
  • En la medida en que un objetivo de este problema es enseñarle cómo aprender un nuevo idioma, tenga en cuenta que estos actos no solo son razonables, según el plan de estudios, sino que se alientan hacia ese fin:
    • Incorporar algunas líneas de código que encuentre en línea o en otro lugar en su propio código, siempre que esas líneas no sean en sí mismas soluciones a los problemas asignados y que cite los orígenes de las líneas.
    • Diríjase a la web o a otro sitio para obtener instrucciones más allá de las propias del curso, para referencias y para soluciones a dificultades técnicas, pero no para soluciones absolutas a problemas del conjunto de problemas o su propio proyecto final.
