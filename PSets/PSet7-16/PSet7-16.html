Honestidad Académica

Problemas

Implemente C$50 Finance en pset7/finance/.

Consejos

  • Asegúrese de utilizar Python 3, no Python 2.
  • Todo el Código que usted escriba deberá ser consistente con la Style Guide for Python Code, también conocido como PEP 8.
  • En la medida en que el objetivo de este problema es enseñarle cómo aprender un nuevo idioma, tenga en cuenta que estos actos no solo son razonables, según el plan de estudios, sino que también se recomiendan para ese fin:
  • Incorporar algunas líneas de código que encuentre en línea o en otro lugar en su propio código, siempre que esas líneas no sean en sí mismas soluciones a los problemas asignados y que cite los orígenes de las líneas.
  • Ir a la web o a otro sitio para obtener instrucciones más allá de las propias del curso, para referencias y para soluciones a dificultades técnicas, pero no para soluciones directas a los problemas del conjunto de problemas o su propio proyecto final.
